# Подсказка по GIT 
Создание репозитория:

```
git init

```

Добавить содержимое файла в индекс

```
git add
```
Записывать изменения в хранилище
```
git commit -m "Massege"
```sh
## Показать коммиты
```
git log
```
## Выводит коммиты красивым списком
```
git log --oneline
```
## Переходит по веткам
```
git checkout <branch name>
```
## Разница между изменениями 
```
git diff
